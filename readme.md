# 基于vue和springboot的电影购票系统10



## 1、项目介绍

基于vue和springboot的电影购票系统拥有两种角色

管理员：影院管理、电影管理、类别管理、场次管理、影厅管理、订单管理、用户管理、角色管理

用户：登录注册、搜索电影、购票、订单管理、个人中心、查看各种信息

## 2、项目技术

后端框架：springboot、Mybatis

前端技术：VUE前后分离

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：无需
- maven版本：无限制
- 硬件环境：Windows
